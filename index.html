一、天道观 — 人之四季

太阳系被周期性统治。地球绕日一周为一年，形成草木的春夏秋冬。人类的「四季」则对应土星绕日周期 — 三十年一周，四个三十年为一个完整循环。

人体受气于天，在出生那一刻形成某种「内模式」，这个内模式与外天地的能量波动互相作用。120年间，总会遇上某个点超过自身结构的稳定性。

衰老未必是必然，只是内模式偏离平衡点的现象。死亡未必是必然，只是自身结构崩溃的现象。如果能越过第一个四季，必然有木质化的跃升。长生，从来都不仅仅是幻想。

二、论命程序

（一）看寿命 →（二）定格局 →（三）定格局高下 →（四）定财富等级 →（五）看婚姻 →（六）看子女 →（七）看老年运

三、寿夭看法

高寿之象

1. 日干得时而专旺成局，贴身伤官或食神旺相健全，吐秀有力。

2. 身旺比劫多且有印，食神或伤官成局，或暗藏食伤局者。

3. 身弱有根于坐下，印绶旺相健全，且贴身相生有情。

4. 身弱印贴身为用神，又有旺官(杀)生用神，又有旺劫或比肩为用神。

5. 八字中和，寒暖适中，用神得力，行运又不相悖。

6. 八字干支无冲克刑克，身心安泰。

7. 日主旺相且得令有气，体神健旺。

8. 身旺财弱有食神扶住财星，精气神渊源流长。

9. 月令没有遭遇日时严重刑冲，根基稳固。

夭命之象

用神浅薄，忌神深重，喜冲而不冲，忌合而反合，行运与用神无情，反与忌神结党。

身旺而克泄全无，身弱而印绶太重，用印而财星坏印，无印而重叠食伤。

金寒水冷而土湿，火炎土燥而木枯。

八字衰弱又见三刑六冲者大凶。日干衰弱，官杀旺相，制官杀的食伤也旺相，大凶。

危险大运判断

1. 身强比劫多无印无官杀，取财为用 → 临冠帝旺运或财星死墓绝运危险。

2. 身强比劫多无印，取官杀为用 → 比劫临冠帝旺运或官杀死墓绝运危险。

3. 身强印多，取财为用 → 比劫临冠帝旺和印旺运危险。

4. 印多日主无根 → 日主长生运最危险。

5. 身强印多少财，食伤为用 → 印枭旺运最危险。

6. 身弱官杀多无印 → 官杀旺运和日主死墓绝运最危险。

7. 化气有余 → 行化神运或生助化神运最危险。

8. 身弱财多无比劫 → 财旺运和日主死墓绝运最危险。

9. 从格命局 → 行克泄耗所从五行运最危险。

10. 四柱天克地冲日柱年柱 → 行犯天克地冲运最危险。

凶运中找凶年

弱命大运通死地 → 死死、死墓、死绝之年。

弱命大运遇墓地 → 墓死、墓墓、墓绝之年。

弱命大运遇绝地 → 绝死、绝墓、绝绝之年。

强命大运遇帝旺 → 旺旺、旺禄、旺生之年。

强命大运遇禄地 → 禄旺、禄禄、禄生之年。

强命大运遇长生 → 生旺、生禄、生生之年。

吉利大运中因刑冲害使某五行突变的流年也凶险。

仇运中凶神劫煞元辰遇两个以上该年为凶。

危险运交运之年和脱运之年往往最凶。

特殊格局口诀

从财从杀运忽挂根，不肯从弃一命归仙。

阳火无根水乡必忌，阴火无根水乡必救。

财逢劫尽、重财破印、劫重见财，皆为死运。

八字一片燥气不见湿土和水，遇水运必死。

金白水清切忌夏生，火伤土制请题铭志。

木火交辉金水伤坏，运行西北死期即到。

活木忌埋根之铁：寅卯未为活木之根，遇申酉丑则根被拔去。

五行生死：遇生怕死，既死怕生；逢旺处生则死处灭，逢死处生必旺处脱。

四、看寿命与生死

1. 日主是否得地得气，太旺无泻、太弱无生。

2. 再找忌神大运，用神或相神被破坏，或某一五行失衡。

3. 确定风险流年：旺而逢生，弱而被克泄。

4. 确定风险真实性：三空相会秘诀验证。

五、定格局

成象：一气专旺、两神成像、化气格等

全形：建禄阳刃格

得局：三合三会成势

取格：四正四生月本气不受损，本气立格，杂气月透干会支立格。

格是人的局限性，某种格就是某一种人；局是人的扩张性，成三合霸道，三会王道。

六、定格局高下

（一）帝王格局

1. 天道得助（天干日主有生助）

2. 地道得局（地支成三会三合局）

3. 人道和合（十神配合成贵格）

4. 时空共振（大运配合无悖逆）

5. 杀伤得用（伤官七杀成格局）

（二）将相格局

天地略有不足，非伤杀得用，十神配合得宜构成贵格，大运配合。吉神成格为相，杀刃成格为将。典型：财官印三奇。

（三）州府格局

财官印三奇缺一而成格，或凶神成格而天地人略有不清或力量不平衡。

（四）县处格局

轻重不足或略有瑕疵，需逢大运去掉格中之病以成就格局。

（五）一般吏员

命局以官印为用而略有格局，有忌神存在，或格多不清。

（六）普通人

格局破损或无格可取，普通人论财官不论格局。

七、定财富等级

（一）顶级富豪

伤官生财格 / 全局流通到财星 / 命局带库(月令最佳) / 生于戌月更佳 / 二十年以上佳运

（二）大富 → 财旺伤官相生，命中带库，十年以上好运

（三）富裕 → 财官格、食神生财格、地支成财局

（四）中产 → 财星可用但根气弱，食神生财而格局不清

（五）工薪 → 命中有财星可用，未成格局

（六）劳禄 → 财太旺而身弱，比劫分财或以比劫取财

（七）贫困 → 财星为忌神且大运再走忌神运，或身旺无依

伤官格佩印主贵，生财主富。

八、看婚姻

看星与宫位，男命看财星，女命看官杀，日支为婚姻宫。宫位助格局即好婚姻。食伤多不在意规则，比劫多性格自我。婚姻对象年龄：食伤财多比自己小，官印重比自己大。

九、看子女

时支为子女宫。女性以食伤为子女，男性以官杀为子女。星为格神或相神，孩子有出息。宫位对日主的作用看关系好坏。

十、看老年运

健康：大运冲提纲阶段需留意；老年行旺运需留意。

破局：需分清是损财还是命局破损而寿终。

旺气及身而止，50岁以后又行旺运者，短寿之命 — 此为老年行运最重要的警示。

命理本质

人之富贵层次取决于天时、地利与人和。格局高下取决于能量级别与流通程度。寿夭取决于日主旺衰、流通与五行中和。六亲关系取决于命局中十神的有情和无情。

生辰八字所揭示的是先天能量配置与天地周期的共振模式。它展示的是趋势与概率，而非铁板钉钉的定数。后天的修养、环境、选择，都在不断重塑生命的轨迹。知命者，方能善用命。

命理揭示的是能量波动的趋势，而非不可逆转的宿命
