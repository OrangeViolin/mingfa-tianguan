一、论命程序

（一）看寿命 →（二）定格局 →（三）定格局高下 →（四）定财富等级 →（五）看婚姻 →（六）看子女 →（七）看老年运

二、看寿命与生死

1. 日主是否得地得气，太旺无泻、太弱无生。

2. 再找忌神大运，用神或相神被破坏，或某一五行失衡。

3. 确定风险流年：旺而逢生，弱而被克泄。

4. 确定风险真实性：三空相会秘诀验证。

三、定格局

成象：一气专旺、两神成像、化气格等

全形：建禄阳刃格

得局：三合三会成势

取格：四正四生月本气不受损，本气立格，杂气月透干会支立格。

格是人的局限性，某种格就是某一种人；局是人的扩张性，成三合霸道，三会王道。

四、定格局高下

（一）帝王格局

1. 天道得助（天干日主有生助）

2. 地道得局（地支成三会三合局）

3. 人道和合（十神配合成贵格）

4. 时空共振（大运配合无悖逆）

5. 杀伤得用（伤官七杀成格局）

（二）将相格局

天地略有不足，非伤杀得用，十神配合得宜构成贵格，大运配合。吉神成格为相，杀刃成格为将。典型：财官印三奇。

（三）州府格局

财官印三奇缺一而成格，或凶神成格而天地人略有不清或力量不平衡。

（四）县处格局

轻重不足或略有瑕疵，需逢大运去掉格中之病以成就格局。

（五）一般吏员

命局以官印为用而略有格局，有忌神存在，或格多不清。

（六）普通人

格局破损或无格可取，普通人论财官不论格局。

五、定财富等级

（一）顶级富豪

伤官生财格 / 全局流通到财星 / 命局带库(月令最佳) / 生于戌月更佳 / 二十年以上佳运

（二）大富 → 财旺伤官相生，命中带库，十年以上好运

（三）富裕 → 财官格、食神生财格、地支成财局

（四）中产 → 财星可用但根气弱，食神生财而格局不清

（五）工薪 → 命中有财星可用，未成格局

（六）劳禄 → 财太旺而身弱，比劫分财或以比劫取财

（七）贫困 → 财星为忌神且大运再走忌神运，或身旺无依

伤官格佩印主贵，生财主富。

六、看婚姻

看星与宫位，男命看财星，女命看官杀，日支为婚姻宫。宫位助格局即好婚姻。食伤多不在意规则，比劫多性格自我。婚姻对象年龄：食伤财多比自己小，官印重比自己大。

七、看子女

时支为子女宫。女性以食伤为子女，男性以官杀为子女。星为格神或相神，孩子有出息。宫位对日主的作用看关系好坏。

八、看老年运

健康：大运冲提纲阶段需留意；老年行旺运需留意。

破局：需分清是损财还是命局破损而寿终。

命理本质

人之富贵层次取决于天时、地利与人和。格局高下取决于能量级别与流通程度。寿夭取决于日主旺衰、流通与五行中和。六亲关系取决于命局中十神的有情和无情。
